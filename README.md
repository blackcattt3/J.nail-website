💅 J.nail Website

J.nail은 네일아트를 단순한 액세서리가 아닌 하나의 예술이자 자기표현의 수단으로 바라보는 브랜드 철학을 웹에 담은 쇼핑몰형 웹사이트입니다.
브랜드 로고, 제품 이미지, 디자인 콘셉트 모두 직접 제작했으며 감각적인 UI 구성과 반응형 레이아웃을 중심으로 제작했습니다.

🧷 프로젝트 소개

이 프로젝트는 HTML, CSS, Bootstrap, FontAwesome을 활용해 제작된 정적 웹사이트입니다.
J.nail의 브랜드 이미지를 중심으로 한 메인 페이지(index.html) 와
회사 정보를 담은 About 페이지(website_info.html) 로 구성되어 있습니다.

심플한 구조와 통일된 블랙 & 화이트 톤 디자인

반응형 레이아웃 (모바일/PC 모두 대응)

Bootstrap Carousel을 이용한 메인 비주얼 슬라이드

Footer 영역에 브랜드 정보 및 메뉴 구성

FontAwesome을 이용한 아이콘 디자인 (user, cart, bar icons 등)
