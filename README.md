<h2>💅 J.nail Website</h2>
J.nail은 네일아트를 단순한 액세서리가 아닌 하나의 예술이자 자기표현의 수단으로 바라보는 브랜드 철학을 웹에 담은 쇼핑몰형 웹사이트입니다.
브랜드 로고, 제품 이미지, 디자인 콘셉트 모두 직접 제작했으며 감각적인 UI 구성과 반응형 레이아웃을 중심으로 제작했습니다.<br>
<br>
↓페이지 바로가기↓
<h2>https://jnail.netlify.app/</h2>
<br>
<h3>📌 프로젝트 소개</h3>
이 프로젝트는 HTML, CSS, Bootstrap, FontAwesome을 활용해 제작된 정적 웹사이트입니다.<br>
J.nail의 브랜드 이미지를 중심으로 한 메인 페이지(index.html)와 회사 정보를 담은 About 페이지(website_info.html) 로 구성되어 있습니다.<br>
<br>
✅ 심플한 구조와 통일된 블랙 & 화이트 톤 디자인<br>
✅ 반응형 레이아웃 (모바일/PC 모두 대응)<br>
✅ Bootstrap Carousel을 이용한 메인 비주얼 슬라이드<br>
✅ Footer 영역에 브랜드 정보 및 메뉴 구성<br>
✅ FontAwesome을 이용한 아이콘 디자인 (user, cart, bar icons 등)<br>
<br>

<h3>✨ 주요 기능</h3>
<h4>1. 네비게이션바 (Navigation Bar)</h4>
✅ 상단 고정 블랙 배경, 로고 및 아이콘 포함<br>
✅ 햄버거 메뉴(fa-bars)를 통한 collapse 토글 (모바일 대응)<br>
✅ FontAwesome 아이콘으로 사용자/장바구니 표시<br>

<h4>2. 메인 슬라이드 (Main Carousel)</h4>
✅ Bootstrap Carousel을 활용한 자동 재생 이미지 슬라이드<br>
✅ 좌우 이동 화살표(fa-chevron-left, fa-chevron-right)로 수동 조작 가능<br>
✅ 브랜드 아트워크 및 제품 이미지 강조<br>

<h4>3. 메인 카드 섹션 (Product Cards)</h4>
✅ Bootstrap Card를 활용한 상품 프리뷰 4종<br>
✅ 제품명: “red & brown”, “wedding”, “black & silver”, “red wine”<br>
✅ 반응형 grid 구성으로 화면 크기에 따라 자동 정렬<br>

<h4>4. About 페이지 (website_info.html)</h4>
✅ 브랜드 로고 및 회사 정보 명시<br>
✅ 주소, 사업자 등록번호, 대표자 이름 등 명확히 정리<br>
✅ flexbox로 수평/수직 정렬 처리<br>

<h4>5. Footer 영역</h4>
✅ COMPANY / COMMUNITIES / SHIPPING 세션 구분<br>
✅ 웹/모바일 버전 각각에 맞춘 레이아웃 분리 (info-list, mobile-info)<br>
✅ 반응형으로 항목 자동 숨김 및 정렬 변경<br>
✅ 하단 저작권 표시 (© 2025 J.nail)<br>
<br>
